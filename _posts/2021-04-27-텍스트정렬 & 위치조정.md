```html
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Insert title here</title>
<style type="text/css">
/* 가로 정렬 - text-align
      세로 정렬 - vertical-align : table에서만 내용정렬
      	따라서 해당 내용을 table에 넣거나
      	line-height을 이용(한줄일때만 사용, 권장하지 않음) */
.a {
	text-align: left;
	background-color: red;
	height: 100px;
	line-height: 50px;
	/* text-indent : 들여쓰기 */
	text-indent: 50px;
	/* letter-spacing : 글자 간격. 권장하지 않음 */
	letter-spacing: -2px;
	/* word-spacing : 띄어쓰기 */
	word-spacing: 10px;
}

.b {
	text-align: center;
	background-color: orange;
	height: 100px;
}

.b1 {
	display: table;
	width: 100%;
	height: 100%;
}

.b2 {
	display: table-cell;
	width: 100%;
	height: 100%;
	vertical-align: middle;
}

.c {
	text-align: right;
}

.d {
	display: inline-block;
	width: 80px;
	height: 80px;
	border: 1px solid #000000;
	/* white-space
	               ㄴ normal : 공백들을 스페이스 한칸으로. 크기보다 클 경우 자동 줄바꿈
	               ㄴ nowrap : 공백들을 스페이스 한칸으로. 크기보다 클 경우 넘침. <br /> 만나야 줄바뀜
	               ㄴ pre : 공백 그대로 적용. 크기보다 클 경우 넘침. <br /> 만나야 줄바뀜
	               ㄴ pre-line : 공백들은 스페이스 한칸 처리하되 엔터는 정상 적용. 크기보다 클 경우 자동 줄바꿈
	               ㄴ pre-wrap : 공백 그대로 적용. 크기보다 클 경우 자동 줄바꿈 */
	white-space: nowrap;
	/* overflow
	               ㄴ visible : 넘치는 내용 다 보여줌
	               ㄴ hidden : 넘치는 내용 감춤
	               ㄴ scroll : 가로 세로 스크롤바 존재
	               ㄴ auto : 넘치는 방향에만 스크롤바 존재 */
	/* overflow-x : x축 지정 시 사용
       overflow-y : y축 지정 시 사용*/
	overflow: hidden;
	/* text-overflow
		      ㄴ clip : 기본
		      ㄴ ellipsis : 넘치면 ...으로 처리 */
	text-overflow: ellipsis;
	/*          가로  세로  블러   크기   색상 */
	box-shadow: 0px 1px 1px 1px black;
}

.e {
	font-size: 50px;
	text-shadow: 0px 0px 4px black;
	color: white;
	height: 300px;
	background-image: url(dog.png);
	background-size: 100px 100px;
	/* background-repeat
	 			  ㄴ repeat : 가로, 세로로 반복
	 			  ㄴ repeat-x : 가로로 반복
	 			  ㄴ repeat-y : 세로로 반복
	 			  ㄴ no-repeat : 반복안함 */
	background-repeat: no-repeat;
	/* background-position : 배경 위치 지정
				  ㄴ % - 배경이미지의 중간지점 위치를 비율로 지정. 단 0%를 잡더라도 이미지는 붙게하여 제공
			          ㄴ px, 외 다른 수치들 - 중간점 위치를 해당 수치에 맞추어 지정)
			          ㄴ top, bottom, center, left, right : 해당지점 방향에 위치시킴 */
	background-position: bottom center;
}

.f {
	height: 100px;
	background-image: linear-gradient(to bottom right, green, blue, purple);
}
</style>
</head>
<body>
	<div class="a">
		Hello 안녕<br />Hi
	</div>
	<div class="b">
		<div class="b1">
			<div class="b2">
				Hello<br />Hi
			</div>
		</div>
	</div>
	<div class="c">Hello</div>
	<div class="d">가나다라마바사아자차카타파하</div>
	<div class="e">Hungry!!</div>
	<div class="f"></div>
</body>
</html>
```

```html
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Insert title here</title>
<style type="text/css">
body {
	margin: 0px;
	font-size: 0pt;
}

.a, .b, .c {
	display: inline-block;
	vertical-align: top;
	width: 100px;
	height: 100px;
}

.a {
	background-color: orange;
}

.p {
	display: inline-block;
	vertical-align: top;
	width: 300px;
	height: 300px;
	background-color: red;
	position: relative;
}

.b {
	/* position: relative;
				- 추가 지정 없을 시 static과 동일 */
	/* position: fixed;
				- 브라우저 크기 기반 고정위치 지정 */
	/* position: absolute;
				- 절대적 위치 지정. 추가지정 없을 시 relative와 동일
				- 조상을 기준으로 절대적 위치 지정 가능(staic인 조상은 안됨. 모든 조상이 static일 경우  body 기준) */
	background-color: purple;
	position: absolute;
	top: 50px;
	left: 50px;
	/* z-index: 값; - 0 ~ n. 클수록 눈에 가까움 */
	z-index: 100;
}

.c {
	background-color: aqua;
	position: absolute;
	top: 100px;
	left: 100px;
	z-index: 50;
}

.popup {
	display: inline-block;
	width: 300px;
	height: 300px;
	background-color: #FFFFFF;
	box-shadow: 0px 0px 1px 1px #444444;
	position: absolute;
	top: calc(50% - 150px); /* 높이의 반만큼 뺌 */
	left: calc(50% - 150px); /* 너비의 반만큼 뺌 */
	z-index: 300;
	font-size: 16pt;
	cursor: pointer;
}

.bg {
	display: inline-block;
	width: 100%;
	height: 100%;
	position: absolute;
	top: 0;
	left: 0;
	background-color: black;
	z-index: 200;
	opacity: 0.6; /* 0.0(투명) ~ 1.0(불투명) */
}
</style>
</head>
<body>
	<div class="a"></div>
	<div class="p">
		<div class="b"></div>
		<div class="c"></div>
	</div>
	<div class="bg"></div>
	<div class="popup">팝업이다.</div>
</body>
</html>
```
